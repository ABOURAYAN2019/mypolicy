Politique de confidentialité

Cette application, ة السياقة المغربية رخصة, respecte la vie privée de ses utilisateurs. Les données personnelles ne sont ni collectées ni partagées avec des tiers, sauf si cela est strictement nécessaire pour le fonctionnement de l'application.

1. Informations collectées

Aucune information personnelle identifiable (nom, email, numéro de téléphone) n'est collectée par l'application.

2. Utilisation des données

Les données saisies par l'utilisateur, telles que les notes scolaires, sont uniquement utilisées localement pour calculer les moyennes et ne sont pas partagées.

3. Droits de l'utilisateur

Les utilisateurs peuvent supprimer toutes les données saisies en désinstallant l'application. Aucune sauvegarde n'est effectuée en dehors de l'appareil.

4. Contact

Pour toute question concernant cette politique de confidentialité, veuillez contacter : moroccodiy@gmail.com

سياسة الخصوصية

يحترم هذا التطبيق رخصة السياقة المغربية خصوصية المستخدمين. لا يتم جمع أو مشاركة أي بيانات شخصية مع جهات خارجية إلا إذا كانت ضرورية لتشغيل التطبيق.

1. المعلومات التي يتم جمعها

لا يتم جمع أي معلومات شخصية يمكن التعرف عليها (مثل الاسم أو البريد الإلكتروني أو رقم الهاتف) عبر التطبيق.

2. استخدام البيانات

يتم استخدام البيانات التي يدخلها المستخدم مثل النقاط فقط محليًا لحساب المعدلات ولا يتم مشاركتها.

3. حقوق المستخدم

يمكن للمستخدمين حذف جميع البيانات عن طريق إزالة التطبيق. لا يتم إجراء أي نسخ احتياطي خارجي.

4. الاتصال

لأي استفسار حول سياسة الخصوصية، يرجى التواصل عبر البريد الإلكتروني: moroccodiy@gmail.com
